=============================================================
             EXECUTIVE RFQ ASSISTANT - TESTING GUIDE
=============================================================
Yeh document aapko complete Procurement Pipeline aur naye 
"View Report" feature ki testing mein guide karega.

-------------------------------------------------------------
METHOD 1: INSTANT UI TESTING (DUMMY DATA)
-------------------------------------------------------------
Agar aap sirf frontend aur UI check karna chahte hain:

1. Apne terminal (VS Code) mein ye command run karein:
   python test_comparison_data.py

2. Apna browser open karein: http://localhost:8069/rfq.html

3. Aapko projects ki list nazar aayegi. Kisi bhi project ke aagay 
   "View Report" par click karein.

4. Aapko direct side-drawer mein Category-wise Comparison 
   Table (AI vs Vendor) nazar aa jayegi.


-------------------------------------------------------------
METHOD 2: FULL PIPELINE TESTING (VIA REAL EMAIL)
-------------------------------------------------------------
Agar aap asal AI pipeline test karna chahte hain (Mail aana -> 
AI se products nikalna -> Live market search hona -> Quote banna):

STEP 1: Ek email compose karein
Apne kisi dusre email address se us email account par mail bhejain 
jo is app ke sath synced hai.

STEP 2: Background Process Ka Wait Karein

   - Items aur quantities extract karega.
   - Live web search kar ke in tools aur PPE ki market price nikalay ga.
   - Ek naya Vendor Quote banayega.

1. Dobara http://localhost:8069/rfq.html par jayein.
